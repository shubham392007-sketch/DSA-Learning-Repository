# Problem-Solving Approach

Every problem added to this repository follows a structured learning process:

## Step 1: Understand the Problem

- Read and analyze the problem statement.
- Identify given inputs and expected outputs.
- Understand constraints and edge cases.

## Step 2: Analyze Possible Solutions

- Think about multiple approaches.
- Compare brute-force and optimized solutions.
- Identify suitable data structures and algorithms.

## Step 3: Implement the Solution

- Write a clean and understandable implementation.
- Follow proper coding practices.
- Add comments wherever required.

## Step 4: Optimize the Solution

- Analyze time complexity.
- Analyze space complexity.
- Improve efficiency whenever possible.

## Step 5: Document Learning

- Record important observations.
- Mention key concepts learned.
- Explain optimization techniques used.
